# Bilingual Output

When writing related work for Chinese-facing notes or drafts:

## Default bilingual mode
1. Keep a concise English section summary or English key terms when helpful.
2. Write the main related-work prose in academic Chinese.
3. Preserve paper titles in English unless the user explicitly wants translated titles.

Recommended shape:

- Follow `references/default-bilingual-template.md` as the default structure.

Rules:
- Chinese prose should read like a paper draft, not a line-by-line translation.
- Keep technical keywords in English when that improves precision.
