# Academic Draft Polish

Polish academic prose while preserving meaning and factual accuracy.

## Workflow

1. Identify the section type and writing goal.
2. Choose an input mode from `references/input-modes.md`.
3. Diagnose the main problems: verbosity, weak logic, casual tone, repetition, unclear claims, or poor transitions.
4. Rewrite the text in a publication-oriented style.
5. Run the edit checklist in `references/edit-checklist.md` mentally before finalizing.
6. Provide a brief edit rationale.
7. When useful, provide a shorter or more formal variant.

## Rules

- Preserve meaning unless the user asks for substantive rewriting.
- Do not invent results, citations, or experimental claims.
- For rebuttals, prefer calm, specific, evidence-oriented language.
- For abstracts, compress aggressively and foreground contribution and evidence.

## Output Pattern

Default output:
1. Polished version
2. Key edits
3. Optional variant

If the user wants a specific delivery format, choose from `references/output-variants.md`.
If the user wants both English and Chinese, follow `references/bilingual-output.md`.
