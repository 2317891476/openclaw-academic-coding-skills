# Role Patterns

Use these roles selectively:

## researcher
Find and summarize sources.

## synthesizer
Merge notes into themes and comparisons.

## writer
Draft user-facing prose.

## reviewer
Check consistency, missing evidence, and structure.

## coder
Implement requested changes.

## tester
Run validation and interpret failures.

Guideline:
- Assign one narrow deliverable per role
- Merge outputs before final response
- Avoid nested delegation unless the task truly requires it
