# Output Format

Use this default card format for each paper:

```markdown
## Paper <n>
- Title:
- URL:
- Year:
- Problem:
- Method:
- Dataset/Benchmark:
- Strengths:
- Limitations:
- Relevance:
- Read priority: high | medium | low
```

Finish with:
- 3-5 trend bullets
- 2-3 gaps or open problems
- a suggested reading order
