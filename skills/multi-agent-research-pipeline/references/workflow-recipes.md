# Workflow Recipes

## Recipe: literature-review
1. Run `research-paper-scout`
2. Save output under `notes/research/...`
3. Hand off paper cards to `related-work-writer`
4. Hand off the draft to `academic-draft-polish`
5. Optionally call `paper-figure-builder`

## Recipe: coding-fix
1. Run `repo-coding-assistant`
2. Extract likely files and validation plan
3. Run `code-implement-test-loop`
4. Save report under `reports/code-loop/...`

## Recipe: paper-section
1. Gather notes
2. Draft section
3. Polish section
4. Produce final clean output
