# Rebuttal Pattern

Recommended shape:

1. Thank the reviewer briefly if appropriate.
2. State the direct answer in one sentence.
3. Provide evidence or clarification.
4. Acknowledge real limitations without sounding evasive.
5. State what will be revised in the paper if relevant.

Example skeleton:

```text
We thank the reviewer for this important point. Our short answer is <answer>. Specifically, <evidence>. We agree that <limitation>, and we will revise the manuscript to clarify <revision>.
```
