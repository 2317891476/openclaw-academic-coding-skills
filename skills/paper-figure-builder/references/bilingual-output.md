# Bilingual Output

When the user wants bilingual figure output:

## Figure body
- Keep diagram nodes primarily in English or standard technical shorthand when that improves clarity.
- Avoid over-translating labels inside the figure if that would make the diagram crowded.

## Figure notes
Provide a bilingual note block:

```markdown
## Figure Note / 图注说明
- EN: <caption or explanation>
- 中文：<academic chinese rendering>
```

Rules:
- The Chinese note should be concise and publication-friendly.
- Keep benchmark names, model names, and standard technical terms in English when appropriate.
- Prefer bilingual captioning over bilingual in-figure text if space is limited.
