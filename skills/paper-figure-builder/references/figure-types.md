# Figure Types

## pipeline
For end-to-end workflows with ordered stages.

## architecture
For modules, components, and their relations.

## comparison
For side-by-side method or system contrasts.

## experiment_flow
For data / training / evaluation process diagrams.

## framework_overview
For a high-level map of a research system.

Guideline:
- Prefer Mermaid for simple pipelines
- Prefer Graphviz for denser graph layouts
- Prefer SVG export for papers
