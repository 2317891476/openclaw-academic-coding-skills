# Pipeline Contract

Default stage order:
1. `research-paper-scout`
2. `related-work-writer`
3. `academic-draft-polish`
4. `paper-figure-builder` (optional)

Expected outputs:
- research note under `notes/research/`
- related-work draft under `notes/related-work/`
- polished bilingual output under `notes/related-work/`
- optional figure source / note artifact

Prefer bilingual academic output by default unless the user asks otherwise.
