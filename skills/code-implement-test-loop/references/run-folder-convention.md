# Run Folder Convention

Save loop artifacts into the same topic-centered coding run folder.

Recommended files:
- `validation-log.json`
- `final-report.md`
- `patch-summary.md` (optional)

Recommended path:
- `research/<topic-slug>/coding/<run-id>/`

Rules:
- One run folder per execution.
- Store both machine-readable and human-readable outputs in the same directory.
