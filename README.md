# OpenClaw Academic & Coding Skills

A curated set of OpenClaw skills for:

- academic paper scouting
- related-work drafting
- academic draft polishing
- paper figure planning and bilingual figure notes
- repository analysis
- implement-test-fix loops
- multi-agent research/coding orchestration

## Included Skills

- `research-paper-scout`
- `related-work-writer`
- `academic-draft-polish`
- `paper-figure-builder`
- `repo-coding-assistant`
- `code-implement-test-loop`
- `multi-agent-research-pipeline`

## Highlights

### Academic workflow
- bilingual output support (English preserved + academic Chinese rendering)
- research-note templates
- related-work structuring rules
- polishing templates for paper sections and rebuttals
- figure note / caption bilingual templates

### Coding workflow
- repository analysis templates
- implement → test → feedback loop
- failure classification
- markdown report rendering helpers

### Orchestration workflow
- handoff contracts between skills
- workflow recipes for literature review and coding-fix pipelines
- multi-agent role patterns

## Repository Structure

```text
skills/

```

## Usage

Place the `skills/` directory under your OpenClaw workspace:

```text
~/.openclaw/workspace/skills/
```

OpenClaw discovers workspace skills from that path.

## Notes

- These skills were designed to work well together as a pipeline.
- The academic skills are optimized for bilingual research output.
- The coding skills are optimized for repository understanding and validation loops.

## Example Pipelines

### Literature review
1. `research-paper-scout`
2. `related-work-writer`
3. `academic-draft-polish`
4. `paper-figure-builder` (optional)

### Coding fix
1. `repo-coding-assistant`
