# Organization Modes

Choose one of these structures:

## by_method
Use when papers differ mainly by model architecture, training paradigm, or optimization strategy.

## by_task
Use when the same broad topic contains distinct subtasks or benchmarks.

## by_timeline
Use when the historical evolution matters and there is a clear progression.

Guideline:
- Default to `by_method`
- Use `by_task` for benchmark-heavy areas
- Use `by_timeline` only when chronology actually helps the argument
