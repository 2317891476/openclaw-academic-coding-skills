# Style Rules

Apply these rules unless the user asks otherwise:

- Prefer precise, low-hype academic wording.
- Replace vague intensifiers with concrete claims.
- Shorten repeated setup language.
- Keep terminology consistent across the paragraph.
- Move the main contribution earlier when possible.
- Use transitions to show causal or comparative logic.
- For rebuttals, answer the reviewer point first, then support it.
- Avoid introducing unsupported certainty.
